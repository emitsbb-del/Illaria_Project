# Illaria_Project
Proyecto elaborado mediante cpp utilizando herramientas como lo es SDL y SDL_ttf. Actualmente este proyecto esta en mantenimiento.

# Illaria

Este es un juego el cual fue inventado como prototipo con el objetivo de dar a conocer al creador y de conocer mas acerca de esta area de programacion

---

## Caracteristicas

- Gameplay 2D
- Sistema de dialogo
- Pixel Art
- Textura apoyadas por IA
- Proyecto escrito con C++

---

## Tecnologias

- C++17
- SDL3
- SDL3_image
- SDL3_ttf
- CMake
- vcpkg
